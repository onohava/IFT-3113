## üöÄ Installation & How to Run

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/onohava/IFT-3113.git](https://github.com/onohava/IFT-3113.git)
    ```
2.  **Open in Unity**
    * Launch **Unity Hub**.
    * Click **Add** and select the cloned `IFT-3113` folder.
    * *Recommended Version:* Unity 2021.3 LTS (or whichever version you used).
3.  **Play**
    * Open the project.
    * Navigate to `Assets/Scenes` and open `MainMenu` or `Level1`.
    * Press the **Play** button in the editor.
